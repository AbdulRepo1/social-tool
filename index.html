Frequently Asked Questions

What size should a Facebook profile photo be?

Facebook displays profile photos at 170 × 170 pixels on desktop (128×128 on smartphones). Select the Facebook Profile Photo preset to resize your image instantly.

What is the correct Facebook cover photo size?

The recommended Facebook cover photo size is 820 × 312 pixels. Images that don't match this ratio are automatically cropped by Facebook, so resize yours first for a pixel-perfect fit.

What size is an Instagram profile photo?

Instagram profile photos are shown at 110 × 110 pixels (in a circle). Upload a high-res square image and use the Instagram Profile Photo preset to resize it perfectly.

Does resizing upload my image to a server?

No. All processing runs entirely in your web browser using HTML5 Canvas. Your images are never sent to any server, making this tool 100% private and safe for personal photos.

What image formats are supported?

You can upload PNG, JPG/JPEG, WEBP, GIF and BMP files. The downloaded output keeps the same format as your original — JPG stays JPG, PNG stays PNG.

Can I use a custom image size?

Yes. Select ── Custom Dimensions ── from the platform dropdown and enter any width and height in pixels (up to 8000 px per side) for a precise custom resize.
